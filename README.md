# 🏨 奥地利大饭店 (Grand Austria Hotel)

电子版桌游，使用 React + TypeScript + Vite 构建。

## 技术栈

- **前端**: React 19 + TypeScript
- **构建**: Vite 6
- **状态管理**: Zustand 5
- **界面**: 深色简约现代风

## 快速开始

```bash
npm install
npm run dev
```

## 游戏规则

### 游戏目标
通过招待客人、建造房间、雇佣员工来获得最高分数。

### 游戏流程

#### 1. 掷骰阶段 🎲
- 当前玩家掷 7 颗骰子
- 可以锁定部分骰子后重掷其余骰子
- 确认后进入选骰阶段

#### 2. 选骰阶段 🎯
- 所有玩家轮流选择骰子（每选一颗切换一次玩家）
- 选择骰子后可以：
  - **取资源**: 骰子点数对应不同资源
    - 1 → 食物×1
    - 2 → 红酒×1
    - 3 → 咖啡×1
    - 4 → 蛋糕×1
    - 5-6 → 金钱×2
  - **选客人**: 骰子点数对应不同颜色的客人
    - 1-2 → 贵族(蓝) / 教士(灰)
    - 3-4 → 政客(黄) / 艺术家(红)
    - 5-6 → 市民(绿)
- 所有骰子选完后进入行动阶段

#### 3. 行动阶段 ⚡
- 每位玩家轮流行动：
  - **招待客人**: 消耗资源，获得分数（需要有空房）
  - **建造房间**: 消耗金钱/资源，获得分数和住宿容量
  - **雇佣员工**: 消耗金钱，获得分数和特殊能力

### 游戏结束
以下条件之一触发游戏结束：
1. 某位玩家招待了 **7 位客人**
2. **所有房间** 被建造完毕

最终分数最高的玩家获胜！

## 游戏数据

| 类型 | 数量 | 说明 |
|------|------|------|
| 客人卡 | 25张 | 5种颜色，每种5张 |
| 房间 | 15个 | 5种颜色，每种3个 |
| 员工 | 8名 | 不同能力 |

## 项目结构

```
src/
├── types/game.ts        # 类型定义
├── data/                # 游戏数据
│   ├── guests.ts        # 客人卡
│   ├── rooms.ts         # 房间
│   └── staff.ts         # 员工
├── game-logic/          # 核心游戏引擎
│   └── engine.ts
├── store/               # 状态管理
│   └── gameStore.ts
└── components/          # UI 组件
    ├── Lobby.tsx        # 开始大厅
    ├── GameBoard.tsx    # 游戏主面板
    ├── DicePanel.tsx    # 骰子面板 (选骰交互)
    ├── ActionPanel.tsx  # 行动面板 (招待/建造/雇佣)
    ├── PlayerBar.tsx    # 玩家信息栏
    ├── PlayerHotel.tsx  # 玩家酒店
    ├── GameLog.tsx      # 游戏日志
    └── WinnerScreen.tsx # 胜利结算
```

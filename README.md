# 🏨 奥地利大饭店 (Grand Austria Hotel)

电子版桌游，基于官方中文规则开发。使用 React + TypeScript + Vite 构建。

## 技术栈

- **前端**: React 19 + TypeScript
- **构建**: Vite 6
- **状态管理**: Zustand 5
- **界面**: 深色简约现代风

## 快速开始

```bash
npm install
npm run dev
```

## 游戏规则（基于官方中规）

### 游戏目标
通过招待客人、建造房间、雇佣员工、推进皇帝轨道来获得最高分数。
游戏进行 **7 轮**，第 3/5/7 轮结束时皇帝计分。

### 游戏组件

#### 客人 (4种颜色)
| 颜色 | 类型 | 偏好房间 | 获取方式 |
|------|------|----------|----------|
| 🔵 蓝色 | 贵族 | 蓝色客房 | 骰子1/5 |
| 🟡 黄色 | 艺术家/政客 | 黄色客房 | 骰子2/6 |
| 🔴 红色 | 市民 | 红色客房 | 骰子3 |
| 🟢 绿色 | 旅客 | 任意客房 | 骰子4 |

#### 房间 (3种颜色)
- 蓝色客房、黄色客房、红色客房，各有 7 个等级
- 等级越高分数越多，建造费用也越高

#### 员工
- 8 名不同能力的员工可供雇佣

### 游戏流程

#### 骰子数量
| 玩家人数 | 骰子数量 |
|----------|----------|
| 2人 | 10颗 |
| 3人 | 12颗 |
| 4人 | 14颗 |

#### 1. 掷骰阶段 🎲
- 起始玩家掷所有骰子
- 可锁定部分骰子后重掷

#### 2. 选骰阶段 🎯
- 所有玩家轮流选择骰子
- 选择后可以：
  - **取资源**: 骰子点数对应资源
    - 1 → 食物×1, 2 → 红酒×1, 3 → 咖啡×1
    - 4 → 蛋糕×1, 5-6 → 金钱×2
  - **选客人**: 骰子点数对应客人颜色

#### 3. 行动阶段 ⚡
- **邀请客人** 💰: 花费金钱邀请大堂的客人到等候区
- **招待客人** 🍽️: 消耗资源满足客人需求（需要有空房）
- **建造房间** 🏗️: 花费金钱获得分数和住宿容量
- **雇佣员工** 👔: 花费金钱获得分数和特殊能力

### 游戏结束
- **固定 7 轮**
- 第 3/5/7 轮结束时 **皇帝计分** (根据皇帝轨道位置)
- 第 7 轮结束后，最高分玩家获胜

## 项目结构

```
src/
├── types/game.ts        # 类型定义
├── data/                # 游戏数据
│   ├── guests.ts        # 20张客人卡 (4色)
│   ├── rooms.ts         # 21个房间 (3色×7)
│   └── staff.ts         # 8名员工
├── game-logic/          # 核心游戏引擎
│   └── engine.ts
├── store/               # 状态管理
│   └── gameStore.ts
└── components/          # UI 组件
    ├── Lobby.tsx        # 开始大厅
    ├── GameBoard.tsx    # 游戏主面板
    ├── DicePanel.tsx    # 骰子面板
    ├── ActionPanel.tsx  # 行动面板 (4Tab)
    ├── PlayerBar.tsx    # 玩家信息栏
    ├── PlayerHotel.tsx  # 玩家酒店
    ├── GameLog.tsx      # 游戏日志
    └── WinnerScreen.tsx # 胜利结算
```
